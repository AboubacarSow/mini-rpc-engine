
# RPC Framework (C#)

A minimal RPC framework built with TCP sockets in C#.  
The project explores the basic concepts behind remote procedure calls.

## Features

- TCP-based transport
- Length-prefixed message framing
- JSON serialization
- Simple RPC request dispatching

## Project Structure 
```
├───Core
│   └───Interfaces
├───Marshaller
│   ├───Interfaces
│   └───Models
│
├───Protocol
└───Transport
```
## Goal

Understand how RPC systems work by implementing a small framework from scratch.
